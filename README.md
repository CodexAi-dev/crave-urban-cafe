# Crave – The Urban Café

A modern, responsive website for **Crave – The Urban Café**, a premium café in Negombo, Sri Lanka.

🌐 Live location: 49/A St Joseph's St, Negombo 11500

## Pages

- **Home** – hero, café experience, photo gallery, signature specials, real Google reviews, and a thank-you message
- **Our Story** – the café's story, philosophy, community, and core values
- **Menu** – view the full menu (PDF)
- **Contact** – address, opening hours, embedded Google Map, and quick-call

## Features

- Fully responsive (desktop, tablet, mobile)
- Photo-background contact hero with parallax
- Masonry photo gallery with "See More" expansion and lightbox
- Real Google Reviews section with rating summary
- Floating WhatsApp chat button
- Branded social links (Facebook, Instagram, TikTok, TripAdvisor)
- Non-intrusive cookie consent banner

## Tech Stack

- **HTML5**, **CSS3**, **Vanilla JavaScript** — no frameworks, no build step
- Google Fonts: Playfair Display, Cormorant Garamond, DM Sans

## Running locally

It's a static site — open `index.html` in a browser, or serve the folder with any local web server (Apache/XAMPP, IIS, Live Server, etc.).

> Note: `web.config` registers the `.webp` MIME type for IIS. On Apache/most static hosts this isn't needed.
